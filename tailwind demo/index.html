Why Tailwind?

Same result. Significantly different workflow.

Vanilla CSS

.spinner {
animation: spin 1s linear infinite;
}
@keyframes spin {
from { transform: rotate(0deg); }
to { transform: rotate(360deg); }
}

Tailwind CSS

// No CSS file needed. Just write HTML.

<div class="animate-spin border-t-sky-500 ..."></div>

animate-spin

// Standard HTML class + Tailwind Animation

<div class="spinner animate-spin"></div>
.animate-spin {
animation: spin 1s linear infinite;
}
@keyframes spin {
from { transform: rotate(0deg); }
to { transform: rotate(360deg); }
}

animate-ping

// Apply utility to the background wave

<div class="ping-wrapper">
<span class="ping-wave animate-ping"></span>
<span class="ping-dot"></span>
</div>
.animate-ping {
animation: ping 1s cubic-bezier(0, 0, 0.2, 1) infinite;
}
@keyframes ping {
75%, 100% { transform: scale(2); opacity: 0; }
}

animate-pulse

// Apply to the main skeleton wrapper

<div class="skeleton-card animate-pulse">
<div class="skel-avatar"></div>
<div class="skel-body">...</div>
</div>
.animate-pulse {
animation: pulse 2s cubic-bezier(0.4, 0, 0.6, 1) infinite;
}
@keyframes pulse {
0%, 100% { opacity: 1; }
50% { opacity: .5; }
}

animate-bounce

// Grabs attention

<div class="bounce-icon animate-bounce">
<svg>...</svg>
</div>
.animate-bounce { animation: bounce 1s infinite; }
@keyframes bounce {
0%, 100% {
transform: translateY(-25%);
animation-timing-function: cubic-bezier(0.8, 0, 1, 1);
}
50% {
transform: translateY(0);
animation-timing-function: cubic-bezier(0, 0, 0.2, 1);
}
}

Customizing Animations

Extend the theme in your tailwind.config.js

animate-wiggle

Wiggle!
keyframes: {
'wiggle': { '0%,100%': { transform: 'rotate(-9deg)' } ... }
}, animation: {
'wiggle': 'wiggle 0.5s ease-in-out infinite'
}
.animate-wiggle { animation: wiggle 0.5s infinite; }
@keyframes wiggle {
0%, 100% { transform: rotate(-9deg); }
50% { transform: rotate(9deg); }
}

animate-float

Float!
keyframes: {
'float': { '50%': { transform: 'translateY(-15px)' } ... }
}, animation: {
'float': 'float 3s ease-in-out infinite'
}
.animate-float { animation: float 3s infinite; }
@keyframes float {
0%, 100% { transform: translateY(0); }
50% { transform: translateY(-15px); }
}
