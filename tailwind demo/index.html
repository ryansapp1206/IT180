Utility-First Animations

Presented by: Ryan Sapp • Monday, 13 April

animate-spin

// 1. Apply utility class

<div class="animate-spin h-24 w-24 rounded-full border-8 border-t-sky-500"></div>
.animate-spin {
animation: spin 1s linear infinite;
}
@keyframes spin {
from { transform: rotate(0deg); }
to { transform: rotate(360deg); }
}

animate-ping

// Apply utility to absolute child

<div class="relative flex h-3 w-3">
<span class="animate-ping absolute h-full w-full bg-rose-400"></span>
<span class="relative rounded-full h-3 w-3 bg-rose-500"></span>
</div>
.animate-ping {
animation: ping 1s cubic-bezier(0, 0, 0.2, 1) infinite;
}
@keyframes ping {
75%, 100% { transform: scale(2); opacity: 0; }
}

animate-pulse

// Apply to skeleton wrapper

<div class="animate-pulse flex space-x-4">
<div class="rounded-full bg-slate-600 h-12"></div>
<div class="h-4 bg-slate-600 rounded w-3/4"></div>
</div>
.animate-pulse {
animation: pulse 2s cubic-bezier(0.4, 0, 0.6, 1) infinite;
}
@keyframes pulse {
0%, 100% { opacity: 1; }
50% { opacity: .5; }
}

animate-bounce

// Grabs attention

<div class="animate-bounce bg-amber-400 p-5 rounded-full">
<svg>...</svg>
</div>
.animate-bounce { animation: bounce 1s infinite; }
@keyframes bounce {
0%, 100% {
transform: translateY(-25%);
animation-timing-function: cubic-bezier(0.8, 0, 1, 1);
}
50% {
transform: translateY(0);
animation-timing-function: cubic-bezier(0, 0, 0.2, 1);
}
}

Customizing Animations

Extend the theme in your tailwind.config.js

animate-wiggle

Wiggle!
keyframes: {
'wiggle': { '0%,100%': { transform: 'rotate(-9deg)' } ... }
}, animation: {
'wiggle': 'wiggle 0.5s ease-in-out infinite'
}
.animate-wiggle { animation: wiggle 0.5s infinite; }
@keyframes wiggle {
0%, 100% { transform: rotate(-9deg); }
50% { transform: rotate(9deg); }
}

animate-float

Float!
keyframes: {
'float': { '50%': { transform: 'translateY(-15px)' } ... }
}, animation: {
'float': 'float 3s ease-in-out infinite'
}
.animate-float { animation: float 3s infinite; }
@keyframes float {
0%, 100% { transform: translateY(0); }
50% { transform: translateY(-15px); }
}
