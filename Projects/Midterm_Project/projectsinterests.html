Recent Software Projects

ytShuffle.com

I recently engineered and deployed ytShuffle.com, a web application that functions as a mathematically random playlist shuffler. To build this, I integrated the YouTube Data API v3 and implemented secure user authentication via OAuth 2.0 to bypass standard algorithm limitations.

The infrastructure is entirely cloud-native, hosted utilizing AWS S3 for storage, AWS CloudFront for global content delivery, and AWS Route 53 for DNS management.
