# IT180 - Web Design Coursework

A comprehensive repository containing all assignments, projects, and coursework completed for IT180. This repository tracks the progression of web development skills from foundational HTML and CSS principles to advanced, responsive, and interactive front-end architecture.

## Course Overview

IT180 focuses on modern web design standards, emphasizing semantic markup, accessible design, and responsive layouts. The coursework spans textbook exercises, sandbox experiments, library presentations, and culminates in a fully interactive, data-driven personal portfolio.
